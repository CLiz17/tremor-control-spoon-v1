## Tremor Control Spoon (Version 1)
A Motorised Sensor Spoon to suppress the hand tremors while eating for the Parkinsonâ€™s or Essential Tremor Patients. The spoon uses Arduino Nano as the controller, MPU6050 as the IMU sensor which has an accelerometer and gyroscope. And a high-precision SG90 servo motor for actuation.
